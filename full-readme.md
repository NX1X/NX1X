### Python Security Scanner
```python
def scan_network(target_range, ports=[22, 80, 443]):
    """Simple network security scanner to check for open ports"""
    results = {}
    for ip in ipaddress.IPv4Network(target_range):
        open_ports = []
        for port in ports:
            sock = socket.socket(socket.AF_INET, socket.SOCK_STREAM)
            sock.settimeout(0.5)
            result = sock.connect_ex((str(ip), port))
            if result == 0:
                open_ports.append(port)
            sock.close()
        if open_ports:
            results[str(ip)] = open_ports
    return results
```

### PowerShell AD Security Check
```powershell
function Get-InsecureUsers {
    param (
        [int]$PasswordAgeDays = 90,
        [switch]$NoMFA
    )
    
    $insecureUsers = @()
    $allUsers = Get-ADUser -Filter * -Properties PasswordLastSet, LastLogonDate, Enabled
    
    foreach ($user in $allUsers) {
        if (-not $user.Enabled) { continue }
        
        $passwordAge = (Get-Date) - $user.PasswordLastSet
        if ($passwordAge.Days -gt $PasswordAgeDays) {
            $insecureUsers += $user
        }
    }
    
    return $insecureUsers
}
```
